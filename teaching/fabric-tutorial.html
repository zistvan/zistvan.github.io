Hyperledger Fabric Tutorial -- How to build applications on a blockchain?

Tutorial offered as part of the chainrEaction 2020 hackathon.

Preparation: Virtual Machine

We have prepared a VM for you to use during the tutorial about Fabric. It can be downloaded from here (1) (the password is "Madrid").

After downloading the image, make sure to adjust the number of CPUs and the size of its allocated memory to values compatible with your computer. The password for the default user is "password". Fabric is installed in the home directory of the user.

If you want to explore the official tutorials, try your hand at this tutorial. The same website will also explain how to install Fabric from scratch (the virtual machine has the dependencies already installed).

Exercises

The slides that provide background on Permissioned Blockchains can be found here (2) and those that describe the exercises to follow are here (3).

You should clone the following repositories for the exercise projects:

  • https://gitlab.software.imdea.org/zistvan-events/fabric-example-gainsierra
